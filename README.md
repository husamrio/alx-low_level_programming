# Low Level Programming :octocat:

This repository contains folders and files related to essential concepts of low-level programming, specifically in the C programming language. Each folder in the repository represents a particular concept, with file names referencing the concept, such as "pointers," "recursion," "malloc," and "free."

Inside each folder, you will find bash scripts and functions related to the specific concept represented by the folder. Some of the functions in the repository are recreations of functions found in the C standard libraries, such as "_strcpy" and "_strlen."

## Environment

The repository was developed in the following environment:

* Language: C
* OS: Ubuntu 14.04 LTS
* Compiler: gcc 4.8.4
* Style guidelines: [Betty style](https://github.com/holbertonschool/Betty/wiki)

## Usage

To use this repository, you will need to have a C compiler installed on your system. You can then navigate to the folder corresponding to the concept you want to learn about and explore the bash scripts and functions provided.

## Contributing
